¿Qué es el internet?

Es un conjunto de "computadoras" que pueden comunicarse entre sí.
Las "computadoras" que almacenan los archivos y permiten que otras computadoras pidan verlos se llaman Servidores

Servidores

Para trabajar(comunicarse) con servidores es necesario utilizar Lenguajes de Programación pensados específicamente para ese fin. Los denominados Lenguajes de Programación back-end (PHP, Java, Phyton, etc)

HTML, CSS y JavaScript

HTML y CSS son Lenguajes que se utilizan para comunicarse con el usuario final.
JavaScript es un Lenguaje que se utiliza para interactuar con el usuario final.
HTML, CSS y JavaScript son Lenguajes del front-end

Estructura de elemento HTML

  • Etiqueta de apertura - Nombre de la etiqueta entre el signo de menor y mayor
    Permite agregar atributos al elemento HTML
  • Contenido - Todo lo que se encuentra entre la etiqueta de apertura y la etiqueta de cierre
  • Etiqueta de cierre - Nombre de la etiqueta entre el signo de menor y mayor. Antes de la primera letra del nombre hay un /

Atributo HTML

Herramienta que me permite distinguir etiquetas del mismo nombre.
Un atributo modifica las etiquetas HTML

Formularios

Herramienta que permite recopilar información

Elementos necesarios para un formulario

  • atributo method (¿cómo va a viajar la información?) - El método de envío de formulario se puede elegir respondiendo la pregunta ¿La información que voy a mandar es sensible?. Como regla general, el método get sólo se usa en las barras de búsqueda
  • atributo action (¿qué va a pasar con esa información?) - El action contiene la ruta al archivo que va a usar los datos recopilados
  • elemento submit (¿cuándo se va a mandar la información?) - Puede ser una etiqueta button o un input con el atributo type="submit"
  • La etiqueta input permite agregar elementos de formulario. El atributo type de la etiqueta indica que tipo de elemento voy a agregar
